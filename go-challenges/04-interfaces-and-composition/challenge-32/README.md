# Challenge 32: Interface Composition

**Difficulty:** ⭐⭐ Medium | **Time:** 20 min

Learn to compose interfaces from smaller interfaces.

## Tasks
1. Define `Reader` interface with `Read() string`
2. Define `Writer` interface with `Write(string)`
3. Compose them into `ReadWriter` interface
4. Implement a type that satisfies `ReadWriter`
