# Challenge 23-30: Error Handling Practice

These challenges cover advanced error handling patterns in Go. Each challenge folder contains:
- README.md with problem description
- main.go with starter code
- main_test.go with test cases
-  SOLUTION.md with complete solution

## Challenges

**23: Errors Package** - Use errors.New, errors.Is, errors.As
**24: Sentinel Errors** - Define package-level error constants
**25: Error Chains** - Unwrap error chains with errors.Unwrap
**26: Panic and Recover** - Handle panics gracefully
**27: Graceful Error Handling** - Best practices for error handling
**28: Error Context** - Add context to errors
**29: Validation Errors** - Collect multiple validation errors
**30: Error Logging** - Log errors effectively

Each challenge builds on error handling concepts. Work through them in order for best results!
