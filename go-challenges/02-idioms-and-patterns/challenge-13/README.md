# Challenge 13: Error as Value

**Difficulty:** ⭐⭐ Medium | **Time:** 15 min

Errors are values in Go, not exceptions! Learn to handle them idiomatically.

## Tasks
1. `SafeDivide(a, b float64) (float64, error)` - return error for division by zero
2. `ChainOperations(x int) error` - chain multiple operations with error checks
3. `MultiError(inputs []string) []error` - collect multiple errors
